Linkitup Publish and enrich Research Data

Linkitup connects to your Figshare account and analyses your (meta)data to propose links to a wide variety of vocabularies and datasets. Linkitup already links to Wikipedia, DBPedia, DBLP, Crossref DOI's, ORCID, and many more.

Try it out » GitHub 

YASGUI Not just another SPARQL GUI

YASGUI is a feature packed web-based SPARQL client interface that supports syntax highlighting and checking, accessing all endpoints, prefix autocompletion, endpoint search and autocompletion, permalinks, persistent application states, query bookmarking, result browsing and configurable requests.

Try it out » More information » GitHub
